Manual do Git

Aqui vai estar a resolução de todos os problemas que enfrentei no git e github. Caso queira procurar por um determinado assunto ou problema, basta ir no input de pesquisa no canto superior direito digitar e depois pressionar enter.

Chave SSH

Para gerar uma chave SSH voce primeiro deve verificar se já existe uma chave SSH usando o terminal do Git-Bash, que por sinal, é instalado junto com o Git, a verificação deve ser feita usando esse comando:

                        
        $ ls -al ~/.ssh

Após ele abrir uma lista de arquivos no terminal, você deve verificar se há na lista três arquivos nescessários para gerar a chave:

                        
        drwxr-xr-x 1 home 197121    0 Aug 17 11:44 ./
        drwxr-xr-x 1 home 197121    0 Aug 27 21:00 ../
        -rw-r--r-- 1 home 197121  748 Aug 17 10:56 id_ecdsa
        -rw-r--r-- 1 home 197121  274 Aug 17 10:56 id_ecdsa.pub
        -rw-r--r-- 1 home 197121  419 Aug 17 11:08 id_ed25519
        -rw-r--r-- 1 home 197121  104 Aug 17 11:08 id_ed25519.pub
        -rw-r--r-- 1 home 197121 3389 Aug 17 10:55 id_rsa
        -rw-r--r-- 1 home 197121  746 Aug 17 10:55 id_rsa.pub
        -rw-r--r-- 1 home 197121  828 Aug 17 11:44 known_hosts
        -rw-r--r-- 1 home 197121   92 Apr  1 15:37 known_hosts.old

Caso seu computador não tenha essas chaves, você poderá adiciona-las usando comandos no Git-Bash.


Primeiro vamos gerar a primeira chave da lista que no caso é a chave ecdsa:

                        
        $ ssh-keygen -t ecdsa -b  521 -f ~/.ssh/id_ecdsa           
                
        -t ecdsa: especifica o tipo de chave como ECDSA.
        -b 521: define o tamanho da chave para 521 bits (o maior suportado pelo ECDSA).
        -f: define o nome do arquivo para a chave.

Em seguida crie a chave id_ed25519, utilizando o mesmo esquema, só que especificandos algumas partes para criar a chave corretamente

                        
        $ ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519

        -t ed25519: especifica o tipo de chave como Ed25519.
        -f: define o nome do arquivo para a chave.

E por último crie a chave a chave rsa:

                        
        $ ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa
                            
        -t rsa: especifica o tipo de chave como RSA.
        -b 4096: define o tamanho da chave para 4096 bits.
        -f: define o nome do arquivo para a chave.

Caso esteja em dúvida se gerou esses arquivos corretamente, você pode repetir novamente o comando de verificação:

        
        $ ls -al ~/.ssh

Se estiver correto, os arquivos id_rsa.pub, id_ed25519.pub e id_ecdsa.pub devem aparecer na lista de arquivos.

Gerando a Chave

Com todos os preparativos prontos nossa chave SSH pode finalmente ser gerada no Git-Bash utilizando o comando:

                
        $ ssh-keygen -t ed25519 -C "seu-email@exmplo.com"

Se seu pc não suporta ed25519, use rsa

                
        $ ssh-keygen -t rsa -b 4096 -C "seu-email@exemplo.com"

Isso irá criar uma chave SSH usando o nome do e-mail como etiqueta

                
        Generating public/private ed25519 key pair.
        Enter file in which to save the key (/c/Users/home/.ssh/id_ed25519):

Agora basta pressionar a tecla enter para passar para a próxima etapa, que no caso é definir uma senha para usar a chave SSH (esse passo é opcional.)

                
        Enter passphrase (empty for no passphrase): [Digete a sena, se quiser.]
        Enter same passphrase again: [Digite novamente]

Após pressionar a tecla enter, o sistema irá criar a respectiva chave juntamente com o e-mail e uma espécie de código QR irá aparecer no terminal do Git-Bash.

Adicionando Chave SSH

Para adicionar a chave na sua conta do GitHub você precisa copiar a chave gerada no arquivo onde foi armazenado a chave, você pode fazer isso manualmente, ou pode fazer com comando, que é o nosso caso:

                
        $ clip < ~/.ssh/id_ed25519.pub

Caso você gerou a chave em rsa, poderá copiar a chave espeficando o arquivo em que foi gerado a chave, que no caso foi em um arquivo rsa.

                
        $ clip < ~/.ssh/id_rsa

com todos os passos finalizados agora é só ir até sua conta do GitHub e adicionar a chave SSH.
